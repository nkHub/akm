检索测试

这里直接调用 `POST /api/markdown-kb/query`。`embedding_model` 是必填项；`reranker_model` 是可选项。建议先看命中的 chunk 是否合理，再决定是否继续调优切片规则或 `top_k`。

问答测试

这里直接调用 `POST /api/markdown-kb/ask`，会先检索知识库片段，再通过 AKM 本地代理生成最终答案，并把引用结果一起返回。`embedding_model` 是必填项，`reranker_model` 是可选项；默认显示 `gpt-4o-mini`，是因为插件配置里的 `chat_model` 默认值就是它；现在这里已经支持直接从当前模型列表中选择覆盖。
