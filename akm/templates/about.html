AI Key Manager

本地 AI API Key 管理代理服务。核心仅提供纯粹的请求转发与审计日志， 协议转换、模型匹配、错误处理等全部由插件接管，可按需启用/禁用。

版本0.1.0
技术栈Python 3.12+ / FastAPI / SQLite / httpx
代理端口8800（默认）
数据目录~/.akm/
仓库github.com/nkHub/akm

Token 统计三种来源

  • CT count_tokens_fallback：Anthropic Messages 在 usage 缺失时通过 messages/count_tokens 回填输入 token。
  • AF usage_fallback_adapter：协议适配器收尾阶段输出的 _last_usage_tokens 回填。
  • EST usage_estimated_light：前两者都不可用时的轻量估算（仅用于可观测，不用于计费）。

审计页模型列会显示 CT/AF/EST 徽标，便于快速判断该条 token 的来源路径。

其他功能

  • - 多供应商 Key 管理 + 自定义供应商代理配置
  • - 429/401/402/5xx 自动故障切换（插件接管）
  • - OpenAI 兼容 API 转发（/v1/chat/completions + /v1/responses + /v1/messages + /v1/models）
  • - 插件系统：六大分类、priority 管道执行、zip 安装、配置读写
  • - Fernet 加密存储 Key + 多重认证头模板支持
  • - Token 用量仪表盘（缓存命中、缓存创建、K/M 格式、1d/7d/30d）
  • - 审计日志（筛选持久化、对话回放、流式收集）
  • - Web 后台管理 + macOS 菜单栏应用
