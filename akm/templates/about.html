AI Key Manager

本地 AI API Key 管理代理服务。集中管理多个 AI 供应商的 API Key， 自动根据优先级选择可用 Key，支持故障切换、请求代理转发及完整审计日志。

版本0.1.0
技术栈Python / FastAPI / SQLite / Click
代理端口8800（默认）
数据目录~/.akm/
仓库github.com/nkHub/akm

功能特性

  • - 多供应商 Key 管理（OpenAI / DeepSeek 等）
  • - 优先级调度与自动故障切换
  • - OpenAI 兼容 API 代理（/v1/chat/completions）
  • - Fernet 加密存储 Key
  • - 完整审计日志与 Token 用量统计
  • - Web 后台管理界面
